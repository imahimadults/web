# シャトルラン録音・合成 Web アプリ

このアプリは、シャトルランの音源を再生しながら、自分の声をマイクで録音し、1つの音声ファイルとして合成・保存できる完全静的な Web アプリです。

## 特徴
- **リアルタイム合成**: マイクの音声とBGMをブラウザ上でミックスして録音。
- **自動音量調整（ダッキング）**: あなたが喋っている間だけ、BGMの音量を自動的に下げて声を聞き取りやすくします。
- **マルチデバイス対応**: iPhone, Android, PC の主要ブラウザで動作。
- **完全静的**: サーバー不要で GitHub Pages などにそのままデプロイ可能。

## 使い方
1.  **音源を選択**: プリセットのリストから選ぶか、「音源をアップロード」ボタンで手持ちのオーディオファイルを選択します。
2.  **録音開始**: 「録音開始」ボタンを押します。マイクの使用許可を求められたら「許可」してください。
3.  **練習・録音**: シャトルランに合わせて声を出しながら録音を行います。
4.  **停止・保存**: 「停止・保存」ボタンを押すと、合成された音声のプレビューが表示されます。
5.  **ダウンロード**: 「録音ファイルを保存」ボタンから、合成済みファイルをダウンロードできます。

## デプロイ方法 (GitHub Pages)
1.  このディレクトリの全てのファイルを GitHub リポジトリ（公開設定）にアップロードします。
2.  リポジトリの **Settings > Pages** を開きます。
3.  **Build and deployment > Branch** で `main` (または `master`) ブランチの `/ (root)` を選択して保存します。
4.  数分待つと、リポジトリに固有の URL (`https://<username>.github.io/<repo-name>/`) が発行されます。

## 使用時の注意点

### 1. HTTPS 環境が必須
ブラウザのセキュリティ制限により、**マイクなどの機能は `https://` で始まる安全な接続環境でのみ動作します**。GitHub Pages は標準で HTTPS 対応されているため、そのまま利用可能です。

### 2. マイク権限
初回起動時に必ずマイクの使用許可をダイアログで許可してください。拒否された場合は設定から許可し直す必要があります。

### 3. イヤホンの推奨
スピーカーで音を出しながら録音すると、スピーカーの音が再びマイクに入り、ハウリングやエコーの原因になります。よりクリアな録音のためには、イヤホン・ヘッドセットの使用を強くおすすめします。

### 4. iPhone (Safari) での動作
- 音量が小さい場合は、側面のサイレントスイッチ（マナーモード）がオフになっていることを確認してください。
- 録音ファイル形式は、iOS のバージョンにより `.mp4` または `.webm` になります。

### 5. 音源ファイルの差し替え方法
`tracks.js` を編集することで、プリセットリストの内容を変更できます。また、`assets/` フォルダ内に独自の音源を配置してパスを指定してください。
